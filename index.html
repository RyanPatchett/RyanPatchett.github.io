Hello! I'm a Game Developer with over 5 years of experience. My journey began in school creating simple games using LUA and Love2D, which evolved into a passion for building robust gameplay systems and AI behaviors.

I thrive on engineering the heartbeat of a game. Whether it's architecting modular ability systems, implementing tight, responsive character controllers, or optimizing complex simulation loops, I am focused on building the core systems that make gameplay feel intuitive and alive.

Technical Arsenal

01. Languages

My core strength lies in C#, which I use daily for gameplay programming and systems architecture. I specialize in writing clean, performant code, utilizing advanced language features and object-oriented patterns to build scalable and robust game mechanics.

02. Engines

My engine of choice is Unity, where I have deep architectural expertise. I leverage the Data-Oriented Technology Stack (DOTS) and the Scriptable Render Pipeline (SRP) for high-performance projects, building modular systems and custom editor tools to push the engine's capabilities to their limit.

03. Tools & Workflow

I maintain a disciplined workflow using Git for robust version control and branching strategies. For asset integration, I utilize Blender for 3D model adjustments and optimizations, while leveraging Photoshop for 2D asset creation, UI elements, and texture manipulation.
